# DigitalCare - Gestion Médicale Collaborative

**DigitalCare** est une application Django de gestion médicale collaborative permettant de gérer :
- Patients
- Personnel médical
- Dossiers médicaux électroniques (DME)
- Rendez-vous
- Hospitalisations
